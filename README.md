# HAVOCAURA
This project is the development of a comprehensive e-commerce platform specializing in the sale of PC parts and pre-built/custom laptops.

## Core Objectives
The primary goal is to create a reliable, accessible online source for computer hardware, catering to both novice and experienced users.

## Key Features and Functionalities
**User-Centric Design:** Prioritizes a user-friendly and intuitive shopping experience.

1. **Custom PC Builder:** Implementation of a seamless "Build a PC" tool to simplify custom computer creation.

2. **Diverse Product Range:** Offering a wide selection of brands and price points, with continuous expansion to include new and in-demand items.

3. **Essential E-Commerce Workflow:** Includes critical functionalities for a smooth purchasing process:

* User Registration and Secure Login

* "Add to Cart" / Shopping Cart Management

* Detailed Invoice Generation

* Secure Online Payment Environment

This platform aims to be the go-to solution for individuals looking to purchase individual components, laptops, or to build complete custom computer systems.
